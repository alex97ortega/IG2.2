
Esta solución de VS2013 v120 (C++ 11) utiliza las libarerias de OGRE3D_1.10.8 para Win32, cons SDL2-2.0.5, freetype-2.6.5 y OpenGL3Plus

Al añadir un nuevo proyecto a la solución recuerda seleccionar C++ vacío y agregar la página de propiedades ProyectosOGRE[_d].props al nuevo proyecto:
Administrador de propiedades Debug/Release-> Agregar hoja de propiedades existente -> ProyectosSDL[_d].props -> Guardar
